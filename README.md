# 📦 Bolsillo Pro — Inteligencia Logística Multiplataforma

¡Bienvenido al repositorio oficial de la Landing Page corporativa de **Bolsillo**! Una suite de ingeniería diseñada para eliminar la incertidumbre operativa, erradicar el robo hormiga y maximizar la rentabilidad de los activos comerciales. 

Este software ha sido diseñado con un enfoque multiplataforma estricto, distribuyéndose como un ejecutable nativo para Windows (`.exe`) y un paquete de alta velocidad para Android (`.apk`).

---

## 🚀 Capacidades de Élite Implementadas

El ecosistema de **Bolsillo** cuenta con herramientas avanzadas desarrolladas para el alto rendimiento empresarial:

1. **Importación Inteligente:** Migración masiva de inventarios desde Excel, CSV o XML con mapeo automático de columnas.
2. **Arquitectura Multi-Bodega:** Supervisión jerárquica y transferencias de stock entre múltiples sedes en tiempo real.
3. **Sincronización Híbrida (Supabase Cloud):** Replicación de datos al milisegundo entre la APK móvil y la aplicación EXE de escritorio.
4. **Escáner Láser de Campo:** Algoritmo de enfoque rápido que transforma la cámara del móvil en un lector de códigos de barras y QR industrial.
5. **Auditoría Inmutable (Log Forense):** Historial permanente de transacciones que registra usuario, acción, fecha y hora exacta.
6. **Valoración Contable Automatizada:** Balances financieros automáticos basados en metodologías PEPS (FIFO), UEPS (LIFO) y Costo Promedio.
7. **Alertas de Stock Crítico:** Notificaciones push inteligentes ante niveles mínimos y módulo avanzado de gestión de mermas.
8. **Reportes Ejecutivos Avanzados:** Exportación instantánea de informes detallados en PDF y Excel para juntas o auditorías fiscales.
9. **Control de Acceso por Roles (RBAC):** Permisos granulares e independientes para Administradores, Gerentes, Cajeros y Operarios.
10. **Respaldos Automatizados:** Copias de seguridad diarias con cifrado simétrico y almacenamiento georredundante en la nube.
11. **Módulo POS Integrado:** Gestión nativa de caja registradora, control de ingresos, abonos y cuentas por cobrar desde el bolsillo.
12. **Arquitectura Offline-First:** Capacidad completa de trabajo en zonas sin cobertura con sincronización automática posterior.

---

## 🛠️ Stack Tecnológico de la Web

- **Frontend:** HTML5 Semántico, CSS3 Avanzado (Variables nativas, Glassmorphism, Responsive Mobile-First) y JavaScript ES6+.
- **Interactividad:** Toggle dinámico de Temas (Claro/Oscuro) y Modales de autenticación integrada listos para producción.
- **Servidor Local:** Entorno estático puro configurado en Node.js para despliegues rápidos en desarrollo.

---

## 💻 Ejecución en Entorno Local

Para previsualizar y probar la landing page localmente utilizando el servidor nativo de Node.js incluido, ejecute los siguientes comandos en su terminal:

```bash
# Iniciar el servidor estático
node server.js
```

El sitio estará disponible por defecto en:
👉 **http://127.0.0.1:8083**
